# Project Roadmap

This document outlines possible directions for expanding **Open Physics 
Simulations** and areas where contributors can help.

The goal is to build a collection of **clear, visual physics simulations** 
that help students and educators develop intuition for physical systems.

---

# Current Simulations

- Photon Random Walk (stellar photon diffusion)
- Black Hole Light Bending (gravitational lensing)
- Electrical Circuit Simulation (RC transient behavior)

---

# Areas for Contribution

## Physics Modeling

Help improve the physical realism of existing simulations.

Possible tasks:

- Implement full **Schwarzschild geodesic integration** for black hole 
lensing
- Improve stellar **density and opacity models** in the photon transport 
simulation
- Extend circuit simulations to include **RLC circuits**
- Add new physical systems

---

## Visualization

Improve clarity and realism of simulations.

Ideas include:

- particle effects
- smoother animation
- improved color schemes
- visual indicators for physical quantities
- optional 3D rendering

---

## Python Development

Improve code structure and performance.

Possible tasks:

- modular simulation framework
- shared utilities across simulations
- performance optimization
- packaging and distribution
- test coverage

---

## Educational Design

Help make the simulations easier to use in classrooms.

Possible improvements:

- example lesson plans
- guided exercises
- parameter exploration modes
- explanation overlays

---

# Future Simulation Ideas

Potential additions to the project:

- Supernova explosion visualization
- Orbital mechanics sandbox
- Wave interference simulator
- Diffusion and heat equation visualizer
- Phase space exploration tools

---

# How to Contribute

If you are interested in contributing:

1. Open an issue to discuss an idea
2. Fork the repository
3. Submit a pull request

Contributors from **physics, computer science, graphics, and education** 
are all welcome.
