# Open Physics Simulations

A collection of **open, educational physics simulations** designed to help students and educators build intuition for physical systems through visualization and interactive modeling.

The goal of this project is to turn **abstract equations into visible behavior** so learners can better understand how physical systems evolve.

These simulations are designed to be:

- **Educational** — suitable for classroom use and self-study
- **Visual** — emphasizing graphical intuition
- **Open** — freely available for exploration and improvement

---

## Current Simulations

### Photon Random Walk
A visualization of photon diffusion inside a star using a random-walk model.  
Demonstrates how radiation slowly escapes dense stellar interiors.

### Black Hole Lensing
A visual simulation of gravitational lensing near a black hole.  
Shows how strong gravity bends light paths and distorts background stars.

### Circuit Simulation
A visualization of electrical circuits showing current flow and basic circuit behavior.

## Project Goals

- Build a collection of **clear, visual physics simulations**
- Provide **free resources for students and educators**
- Encourage exploration of physics through computation
- Maintain scientifically meaningful models while prioritizing intuition
