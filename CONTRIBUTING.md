# Contributing

Thank you for your interest in contributing to **Open Physics 
Simulations**.

This project aims to build clear, visual simulations that help students 
and educators develop intuition for physical systems.

Contributors from physics, computer science, visualization, and education 
are all welcome.

---

## Ways to Contribute

You can help by:

• improving physics models  
• improving visualizations  
• improving Python architecture  
• suggesting new simulations  
• helping design classroom uses

See `ROADMAP.md` for current areas where help is needed.

---

## Contribution Process

1. Open an **issue** to discuss your idea or proposed change.
2. Fork the repository.
3. Create a new branch for your changes.
4. Submit a **pull request**.

Please include a short explanation of what the change does and why.

---

## Goals of the Project

This project prioritizes:

• clarity and intuition  
• scientifically meaningful models  
• educational usefulness  
• open collaboration

---
