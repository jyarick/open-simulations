# Planet Formation Simulator

Simple physics-based planet formation toy models written in Python.

## Versions

### planet_sim.py
Original experiment (3D attempt)

### planet_sim_turtle.py
Stable 2D simulation using Turtle graphics with:

- gravitational star potential
- gas drag
- particle collisions
- accretion and fragmentation
- dust generation
- protoplanet growth visualization

## Run

python planet_sim_turtle.py
